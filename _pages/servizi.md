## Sviluppo Backend Java

Progettazione e sviluppo di applicazioni server-side con **Java** e **Spring Boot**: API REST, layer di business logic, integrazione con sistemi esistenti, AspectJ per logging e monitoring trasversale.

## Sviluppo Frontend

Sviluppo di interfacce web enterprise con **AngularJS** e JavaScript. Ottimizzazione performance UI su grandi volumi di dati (virtual scrolling, ottimizzazione digest cycle), rich text editor, gestione tabelle complesse.

## Database & SQL Server

Progettazione e ottimizzazione di query complesse, stored procedure, migration di schemi. Esperienza con **SQL Server** e ORM **MyBatis**.

## Integrazione Sistemi

Integrazione con sistemi aziendali esistenti: LDAP/Active Directory, sistemi di identity management, generazione documenti PDF con **JasperReports**, report con **SSRS**.

## Monitoring & Logging

Implementazione di logging strutturato con **ELK Stack / Kibana**, propagazione del contesto MDC in architetture async, tracciabilita completa delle operazioni.

## Documentazione & Release Management

Redazione di documentazione tecnica, release notes, analisi OWASP e SBOM per la sicurezza delle dipendenze. Gestione del ciclo di rilascio con **IBM RTC** e **Git**.
